# Python & ML Basics

Python, pandas, scikit-learn 기초 학습 및 실습 노트북 모음.

## 폴더 구성
- `python-basics/` — Python 기초 문법 실습 (문자열, 함수, 기초 예제)
- `pandas-numpy/` — numpy/pandas 기초 및 데이터 분석 교재 챕터 실습 (Chapter6~14)
- `sklearn-ml/` — scikit-learn 분류/회귀 실습: 데이터 전처리, 교차검증, 타이타닉 생존자 예측, 정확도·F1 Score 등 평가지표
- `bootcamp/` — 부트캠프(데청캠) 실습 코드
- `data/` — 타이타닉 실습 데이터
